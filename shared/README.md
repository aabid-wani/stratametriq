# @stratametriq/shared — Core TypeScript Interfaces & Graph Models

> **StrataMetriq helps engineering teams understand software architecture, predict the impact of code changes, and prevent risky deployments before they happen.**

**@stratametriq/shared** provides the universal TypeScript type definitions, graph node/edge models, risk severity definitions, and custom architecture governance interfaces used across all StrataMetriq packages (`@stratametriq/scanner`, `@stratametriq/cli`, and `stratametriq-extension`).
